# YouTube Data API Project

## Description
Use the FastAPI to enter a video ID and view statistics of certain Youtube videos. See how many days it took to become viral, like ratio, etc. 

## Installation
To set up this project on your local machine, follow these steps:

1. Clone the repository and open directory through git bash or command prompt

2. pip intsall -r requirements.txt

3. uviocorn main:app --reload

4. To access API: Go to http://127.0.0.1:8000/docs

5. In /videos{video_id} Click on Try it out, and enter in a video_id from the df in the Youtube Consumption dataframe

6. In /upload/ Upload Video - You can upload a new video to the dataframe
